# Jeu de l'Oie

This _goose game_ is for playing with my kids.

__Oie__ is implemented in HTML5/JavaScript and propulsed with [Crystal](https://crystal-lang.org/) programming language.

## Requirements

You need a 30 squares printed board.

Have fun!
